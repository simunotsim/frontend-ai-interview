# InterviewAI Frontend

This is the frontend application for InterviewAI, an AI-powered interview and career platform.
Built with React and Tailwind CSS.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

## Application Routes

Once the local server is running (by default on port 3000), you can access the different authentication flows:

- **Student / Candidate Flow:** [http://localhost:3000/student](http://localhost:3000/student) (or simply [http://localhost:3000/](http://localhost:3000/))
- **Company / Employer Flow:** [http://localhost:3000/company](http://localhost:3000/company)

Each flow uses a responsive split-screen layout with tailored form fields, custom AI illustrations, and specific marketing content for that user role.
